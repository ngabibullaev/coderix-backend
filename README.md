## 🚀 Быстрый старт

### Установка и запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/ngabibullaev/coderix.git

# Перейдите в папку проекта
cd coderix

# Установите зависимости
npm install

# Запустите в режиме разработки
npm start
